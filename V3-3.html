일반적으로 상하 또는 주종 관계로 표현되는 서비스를 공급하는 제공자를 서버(Server)라 하고, 서비스를 제공받는 고객 즉, 클라이언트라는 관계가 형성된다.

웹(Web)에서 한정한다면, 웹 브라우저 (크롬,익스플로어,사파리 등)가 클라이언트라 할 수 있고, 웹 서버 ( 아파치(Apache), 엔진엑스(nginx), 마이크로소프트사의 IIS(Internet Information Services) 등)가 서버라 할 수 있다.

웹 브라우저, 즉 크롬이나 익스플로어 등에서 내가 원하는 웹페이지 주소를 입력하는 행동은 이 웹 브라우저가 웹 서버에 요청하는 것을 의미하고, 이 웹서버는 그 입력된 주소를 찾아 응답을 하면 웹 브라우저에 그 웹페이지가 나타난다는 것이다. 이것이 인터넷이 작동하는 기본 원리라고 한다. 여기서 요청을 하는 웹 브라우저를 클라이언트, 즉 요청을 하는 고객이라는 비유로 해서 클라이언트라고 정했다고 한다. 그리고 응답하는 웹서버를 사업자라는 개념으로 서버라고 정했다고 한다.
따라서, 웹 개발을 위한 언어를 구분한다면, 클라이언트 즉, 웹 브라우저 관련 언어는 HTML, CSS, JavaScript 등이 있고, 웹 서버 관련 언어는 PHP, Node.js, MySQL 등이 있다. 이 둘에 대하여 프론트엔드(Front-end)와 백엔트(Back-end)로 구분하기도 한다.
